# Local Tourist Day-Visit Planner and Information System

This is a web-based system developed to help users plan one-day visits to tourist places within a 25 km radius from Moratuwa.

## Features
- Browse tourist places
- Search and filter places
- View place details
- Add places to visit planner
- Admin can manage places

## Technologies Used
- HTML, CSS, JavaScript
- PHP
- MySQL
- XAMPP

## How to Run
1. Install XAMPP
2. Place project in htdocs folder
3. Start Apache and MySQL
4. Open browser and go to:
   http://localhost/tourist_planner
